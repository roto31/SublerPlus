# Troubleshooting

- Build: ensure Swift 5.9+, keys set.
- Metadata not updating: check file permissions; Activity/Jobs panes.
- Ambiguity: resolve in the sheet; clear cache in Settings.
- WebUI: use http://127.0.0.1:8080; set token; ensure assets present.
- Providers: retry/backoff; verify keys (TPDB/TMDB/TVDB/OpenSubtitles); watch rate limits.
- Subtitles: verify OpenSubtitles key (RapidAPI), check language code (ISO 639-2).
- Watch folders: ensure correct permissions, MP4/M4V/MOV files, monitor is active.
- NFO/Output: verify output directory settings, check folder permissions.
- Logging: Console.app subsystem `com.sublerplus.app`; secrets scrubbed.
