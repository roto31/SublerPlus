# Troubleshooting

- Build: ensure Swift 5.9+, keys set.
- Metadata not updating: check file permissions; Activity/Jobs panes.
- Ambiguity: resolve in the sheet; clear cache in Settings.
- WebUI: use http://127.0.0.1:8080; set token; ensure assets present.
- Providers: retry/backoff; verify keys; watch rate limits.
- Logging: Console.app subsystem `com.sublerplus.app`; secrets scrubbed.
