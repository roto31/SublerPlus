# SublerPlus How-To Guide

> **Note**: This is a summary version. For the complete guide, see [docs/HOW_TO_GUIDE.md](../docs/HOW_TO_GUIDE.md)

## Quick Start

### Installation
1. Build: `swift build`
2. Run: `swift run SublerPlusApp`
3. Configure API keys in Settings

### Basic Workflow
1. Add files (drag-drop or file picker)
2. Click "Enrich" to add metadata
3. Review and apply results
4. Monitor progress in Jobs section

## Key Features

### Drag-and-Drop Metadata Search
- Drop files onto Advanced Search pane
- Automatic metadata extraction
- Sidebar displays file metadata
- Auto-populated search fields
- Apply metadata with artwork selection

### TV Show Metadata
- Extracts TV show info from MP4 atoms
- Shows season/episode information
- Displays in sidebar when available

### Muxing and Presets
- Use built-in or custom presets
- Import/export presets
- Track selection and conversion
- Audio and subtitle conversion

## Troubleshooting

### Common Issues
- **Files not adding**: Check format (MP4, M4V, MOV)
- **No metadata found**: Verify API keys and internet
- **Metadata not applying**: Check file permissions
- **Artwork not loading**: Clear cache or refresh
